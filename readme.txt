== Description ==

Allow to create custom lists of content based on post types, taxonomies and others parameters. It uses a lightbox and generate a shortcode. Easy to use.

== Installation ==

1. Upload and activate the plugin
2. That's all

== Screenshots ==
1. The shortcode generator
2. The shortcode in a post
3. The generated list

== Usage ==

1. Add a post and click on the Custum Content List button on your TinyMCE.
2. Select your post type and if your want select your taxonomy and your term
3. You can select the list order and customize the list title
